Usando links externos

Voce pode acessar o meu repositorio publico no GitHub

Voce tambem pode acessar o meu canal gratuito no YouTube

Fique atento ao termo < target="_blank" rel="external"> que serve para abrir o link em uma nova aba, sem ele o link abre na mesma aba(não recomendado, pois o usuario pode fechar a aba ou não voltar para o seu site)

Existem vários tipos de valores para o atributo target que são:

    • _self: vai abrir o link na janela ou frame atual(padrao)
      _parent: vai abrir o link na janela ou frame pai(similar ao _top)
      _top: vai abrir o link na janela ou frame superior
      _blank: vai abrir o link em uma nova janela ou frame

    • Já no caso do rel ele vai indicar o tipo de relacionamento que o link tem com o site atual, ele apresenta os seguintes valores:

      external: Indica que o documento referenciado não faz parte do mesmo site que o documento atual.
      bookmark: URL permanente usada para favoritos.
      noopener: Garante que qualquer contexto de navegação criado ao seguir o link não tenha um contexto de abertura.
      noreferrer: Garante que o referenciador não seja enviado ao seguir o link.
      follow: indica que o link vai abrir na mesma aba
      nofollow: Indica que o link não deve ser seguido por motores de busca.
      next: Fornece um link para o próximo documento em uma série.
      prev: Fornece um link para o documento anterior em uma seleção.
      search: Fornece um link para uma ferramenta de busca para o documento.
      author: Indica o autor do documento.
      tag: Indica o tag do documento referenciado.
