Os media types em HTML são usados para especificar o tipo de conteúdo que está sendo enviado ou recebido. Aqui estão alguns dos mais comuns e suas funções:

  • text/html: Usado para documentos HTML.
  • text/css: Usado para folhas de estilo CSS.
  • text/javascript: Usado para scripts JavaScript.
  • image/jpeg: Usado para imagens JPEG.
  • image/png: Usado para imagens PNG.
  • audio/mpeg: Usado para arquivos de áudio MP3.
  • video/mp4: Usado para arquivos de vídeo MP4.
  • application/json: Usado para dados no formato JSON.
  • application/xml: Usado para dados no formato XML.
Esses tipos de mídia ajudam os navegadores e servidores a entenderem como processar e exibir o conteúdo corretamente12.
